# DimensionCraft

DimensionCraft is a Minecraft Forge mod that adds exciting new dimensions and adventures to the game. Explore mysterious worlds, face unique challenges, and unlock powerful abilities!

## Features

- Multiple new custom dimensions with unique biomes and mobs  
- New crafting recipes and items to aid your journey  
- Engaging lore and quests tied to the dimensions  
- Smooth cutscenes and immersive gameplay events  

## Installation

1. Ensure you have Minecraft Forge installed for the correct Minecraft version.  
2. Download the latest DimensionCraft mod `.jar` file from the [Releases](https://github.com/Tabbeys/DimensionCraft/releases) page.  
3. Place the `.jar` file into your Minecraft `mods` folder.  
4. Launch Minecraft with the Forge profile and enjoy!
